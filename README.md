# ChunkGlow-QD-Series-
The ChunkGlow QD Series is a Series of Codecs and Formats made by the ChunkGlow Team. This includes QD-3, QD-6, QD-9 and Advanced Formats made by the team

#Introduction
The ChunkGlow QD-Series Is a List of Media Codecs and Data codecs that contains the main QD-3, QD-6, QD-9 and others like QD-L, QD-S, QD-i, QD-C, QD-F and QD-R
